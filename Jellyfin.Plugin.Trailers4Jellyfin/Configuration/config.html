Trailers4Jellyfin

Help

Automatically downloads trailers for upcoming and recently released movies not in your library from TMDB/YouTube. Save them to a folder, then point Cinema Mode's Pre-Roll library at that folder to get a real cinema experience — random trailers for films you haven't seen before your movie starts.

A free TMDB API key is required. Get one here →

TMDB
Your TMDB v3 API key from themoviedb.org/settings/api.

Trailer Sources

Choose which TMDB lists to pull trailers from. Enable multiple for more variety.

Date Range
Applies to all sources. "Now Playing" and "Upcoming" already have tight date windows set by TMDB, but this provides an additional filter.

Download Settings
Where trailers are saved. After the task runs, create (or refresh) a Jellyfin Movies library pointing at this folder, then set it as the Trailer Pre-Roll Library in the Cinema Mode plugin settings.
Maximum number of trailers to download each time the task runs. Default: 20.
How many pages to fetch from each TMDB source (20 movies per page). Increase if you have many movies already in your library and need more candidates. Default: 3.
Trailers for movies you already own won't be downloaded. Enabled by default.
If a file already exists for a movie, don't re-download it. Enabled by default.

Advanced
Full path to yt-dlp. Required for 1080p quality. Also needs ffmpeg on the system PATH. Leave blank to use the built-in downloader (720p max, zero extra tools).
